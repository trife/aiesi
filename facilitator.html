Before the room fills up

  • Have everyone open materials.html as they arrive - it confirms their browser can run the demos and lets them test their camera, so nobody hits a surprise mid-activity.
  • Project the hub page (index.html) so people can find their way back to it.

Part 1 - Do the AI science (~75 min)

Segment 1 - Browser demos (~35 min)

Site: demos/index.html

  • Frame it: "You write the rules by hand - you are the model."
  • Suggest ~8 min per demo: canopy cover → count the objects → greenness/health index → edge detection.
  • For each: adjust the sliders, then use that demo's "Try this → Now check it" panel as the built-in prompt - no separate handout needed.

Segment 2 - Teachable Machine (~25 min)

Site: teachable-machine.html

  • Frame it (2 min): "Now flip to learned from examples - you show it labeled leaves, it finds the rule."
  • Train (10 min): two classes, 10–20 photos each. Walk the room - watch for badly imbalanced classes (18 vs. 4) as a live teaching moment.
  • Test (5 min): normal conditions, should work well and feel impressive.
  • Break it (8 min): edge cases on purpose - blur, weird angle, a hand, wrong plant entirely. Point out the confidence score rarely drops.

Segment 3 - Name the contrast (~15 min)

Site: contrast.html

  • Pull up the two-column recap. Ask: where did each approach break for you specifically?

Part 2 - Discuss the science (~75 min)

Segment 1 - Adapt the trust check (~30 min)

Site: downloads/trust-check.pdf

  • Hand out (or have people download) the trust-check worksheet.
  • Have each teacher rewrite the six questions for their own subject - a biology teacher for a cell-counting AI, a chemistry teacher for a spectra classifier.
  • Working solo or in pairs is fine; the goal is a first draft, not a polished worksheet.

Segment 2 - Design one activity (~30 min)

Site: downloads/activity-template.pdf

  • Small groups (3–4). Each group builds or retrofits one classroom exercise.
  • Constraint to repeat out loud: AI does the grunt work, but the graded skill is judgment - not "did you get the AI's number."
  • Push groups to name specifically what a student would be graded on.

Segment 3 - Share-out (~15 min)

  • One minute per group: what's the activity, what's the graded skill.
  • Close on the throughline: today's tools change, but "where's the ground truth, and does this answer earn your trust" doesn't.
